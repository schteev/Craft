# Craft HTML/JS Game

A browser-based block game demo with Minecraft-style terrain, water, food, wildlife, and day/night mechanics.

## Files

- `index.html` — game UI, controls, and canvas
- `styles.css` — visual layout and presentation
- `script.js` — world generation, logic, physics, entities, and day/night cycle

## Controls

- `A` / `←`: Move left
- `D` / `→`: Move right
- `W` / `↑` / `Space`: Jump
- `C`: Crawl
- Click: Place the selected block
- `Shift+Click`: Remove a block

## Features

- Water with fish, sharks, turtles, stingrays, coral, seaweed, and crabs
- Food blocks like cake, soup, and chips
- Tilted glass and laser blocks
- Day/night cycle with zombies appearing at night
- Land animals including cows, chickens, frogs, sheep, pigs, and threshers (thresher sharks)
- Birds flying around
- Blocks: chests, trees, branches, lilies, buckets, television, computers, camera, pots, and bushes

## Optional local server

For the best experience, serve the files from a local server:

```bash
npx http-server .
```
